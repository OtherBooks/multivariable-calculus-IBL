EXTRA 1 You'll want a computer to help you construct the graphs, particularly \(h\text{.}\) \bmw{Please use the Mathematica introduction in Brainhoney.} You could use Wolfram Alpha (use the links in the function chapter if you forgot how to graph). Consider three functions:

  • \(f(x,y)=9-x^2-y^2\)

  • \(g(x,y)=2x-y\)

  • \(h(x,y)=\sin x\cos y\)

Hint

If you need a reminder of how to draw level curves look back at exercise 7.4.2

  1. Compute \(\vec \nabla f(x,y)\text{.}\) Then draw both \(\vec \nabla f\) and several level curves of \(f\) on the same axes. EXTRA 2 See Sage. You can modify these commands to help in the plots below too.

  2. Compute \(\vec \nabla g(x,y)\text{.}\) Then draw both \(\vec \nabla g\) and several level curves of \(g\) on the same axes.

  3. Compute \(\vec \nabla h(x,y)\text{.}\) \valpo{You do not need to plot this one, I'll show you the plot.} \bmw{Then draw both \(\vec \nabla h\) and several level curves of \(h\) on the same axes.}

  4. What relationships do you see between the gradient vector field and level curves?

\bmw{When you present in class, be prepared to provide rough sketches of the level curves and gradients of each function.}
