Exercise8.4.2

Suppose we know that \(\ds f'(x) = \frac{\sin(x)}{2x^2+3}\) and \(g(x)=\sqrt{x^2+1}\text{.}\) Notice we don't know \(f(x)\text{.}\) EXTRA 1 Not knowing a function \(f\) is actually quite common in real life. We can often measure how something changes (a derivative) without knowing the function itself.

  1. State \(f'(x)\) and \(g'(x)\text{.}\)

  2. State \(f'(g(x))\text{,}\) and explain the difference between \(f'(x)\) and \(f'(g(x))\text{.}\)

  3. Use the chain rule to compute \((f\circ g)'(x)\text{.}\)
