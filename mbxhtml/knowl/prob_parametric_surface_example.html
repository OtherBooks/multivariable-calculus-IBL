Exercise7.3.1

EXTRA 1 See Sage or Wolfram Alpha. \larsonfive{More practice in Larson 15.5:1–6.} The jet from problem 7.2.2 is actually accompanied by several jets flying side by side. As all the jets fly, they leave a smoke trail behind them (it's an air show). The smoke from one jet spreads outwards to mix with the neighboring jet, so that it looks like the jets are leaving a rather wide sheet of smoke behind them as they fly.

The position of two of the many other jets is given by \(\vec r_3(t)=(3\cos t, 3\sin t, t)\) and \(\vec r_4(t)=(4\cos t,4\sin t,t)\text{.}\) A function which represents the smoke stream is \(\vec r(a,t)=(a\cos t, a\sin t, t)\) for \(0\leq t\leq 4\pi\) and \(2\leq a\leq 4\text{.}\)

  1. What are \(n\) (inputs) and \(m\) (outputs) when we write the function \(\vec r(a,t)=(a\cos t, a\sin t, t)\) in the form \(\vec r\colon {\mathbb{R}}^n\to {\mathbb{R}}^m\text{?}\)

  2. Start by graphing the position of the three jets. For \(t=0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}\) plot the position of each jet:

    1. \(\vec r_2(2,t)=(2\cos t, 2\sin t, t)\)

    2. \(\vec r_3(3,t)=(3\cos t, 3\sin t, t)\)

    3. \(\vec r_4(4,t)=(4\cos t, 4\sin t, t)\)

  3. We said in the initial problem statement that the smoke spreads and merges. So we really want to plot \(\vec{r}(a,t)=(a\cos t, a\sin t, t)\) for \(2\leq a \leq 4\) at each \(t\) value.

    1. Describe how would this modify your graph from the previous part?

    2. Let \(t=0\) and graph the curve \(r(a,0)=(a,0,0)\) for \(a\in[2,4]\text{.}\)

    3. Repeat this for \(t=\pi/2,\pi,3\pi/2\)

  4. Describe the resulting surface.
