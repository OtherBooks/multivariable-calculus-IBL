Consider the surface \(S\) given by \(z=9-x^2-y^2\text{,}\), for \(z\geq 0\text{.}\). A different parametrization of this surface is \begin{equation*} \vec r(r,\theta) = (r\cos\theta,r\sin\theta,9-r^2), \text{ where } 9-r^2\geq 0. \end{equation*}

  1. Give a set of inequalities for \(r\) and \(\theta\) that describe the region \(R_{r\theta}\) over which we need to integrate. The inequalities you give should be in a form that you can use them as the bounds of a double integral.

  2. Find \(d\sigma = \left|\vec r_r\times \vec r_\theta \right|drd\theta\text{.}\)

  3. Set up the surface integral \(\iint_S d\sigma\) as an iterated double integral over \(R_{r\theta}\text{.}\)
