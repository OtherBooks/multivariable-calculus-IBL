The number \(|r|\) is called the Jacobian of \(x\) and \(y\) with respect to \(r\) and \(\theta\text{.}\). If we require all bounds for \(r\) to be nonnegative, we can ignore the absolute value. If \(R_{xy}\) is a region in the \(xy\) plane that corresponds to the region \(R_{r\theta}\) in the \(r\theta\) plane (where \(r\geq 0\)), then we can write \begin{equation*} \iint_{R_{xy}} f(x,y) dxdy = \iint_{R_{r\theta}} f(r\cos\theta,r\sin\theta) r\ drd\theta. \end{equation*}
