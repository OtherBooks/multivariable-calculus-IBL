EXTRA 15 \larsonfive{See Larson 11.6:7–16.}Consider the function \(w=f(x,y,z)=x^2+z^2\text{.}\) This function has an input \(y\text{,}\) but notice that changing the input \(y\) does not change the output of the function. This means that when \(y=0\) you can draw one curve. When \(y=1\text{,}\) you should draw the same curve. When \(y=2\text{,}\) again you draw the same curve, etc.

  1. Draw a graph of the level surface \(w=4\text{.}\)

  2. Graph the surface \(9=x^2+z^2\) (so the level surface \(w=9\)).

  3. Graph the surface \(16=x^2+z^2\text{.}\)

This kind of graph is called a cylinder, and is important in manufacturing where you extrude an object through a hole.
