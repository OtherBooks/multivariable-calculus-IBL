EXTRA 3 \bmw{See 14.8 for more practice.} Consider the curve \(xy^2=54\) (draw it). The distance from each point on this curve to the origin is a function that must have a minimum value. Find a point \((a,b)\) on the curve that is closest to the origin.

Hint

The distance to the origin is \(d(x,y)=\sqrt{x^2+y^2}.\) This distance is minimized when \(f(x,y) = x^2+y^2\) is minimized. So use \(f(x,y)=x^2+y^2\) as the function you wish to minimize. What's the constraint \(g(x,y)=c\text{?}\)
