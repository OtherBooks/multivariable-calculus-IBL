Item10.4.3.3

  • We'd like to find a constant height \(\bar f\) so that the area under \(f\text{,}\) above \(C\text{,}\) is the same as area under \(\bar f\text{,}\) above \(C\text{.}\) This height \(\bar f\) is called the average value of \(f\) along \(C\text{.}\) EXTRA 7 \bmw{Please read Isaiah 40:4 and Luke 3:5. These scriptures should help you remember how to find average value. } \valpo{ Please read Isaiah 40:4 and Luke 3:5. These scriptures should help you remember how to find average value. } Explain why the average value of \(f\) along \(C\) is \begin{equation*} \bar f = \frac{\int_C f ds}{\int_C ds}. \end{equation*} Connect this formula with the ways of thinking about averages from Exercise 10.4.1. Hint

    The area under \(\bar f\) above \(C\) is \(\int_C \bar f ds\text{.}\) The area under \(f\) above \(C\) is \(\int_C f ds\text{.}\) Set them equal and solve for \(\bar f\text{.}\)
