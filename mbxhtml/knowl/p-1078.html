The vector \(\vec N\) is a unit vector in the same direction as the curvature vector. Since \(\dfrac{d\vec T}{dt}\) and \(\dfrac{d\vec T}{ds}\) point in the same direction, we could have written \(\vec N = \dfrac{d\vec T/ds}{|d\vec T/ds|} = \dfrac{\vec \kappa}{\kappa}\text{.}\).
