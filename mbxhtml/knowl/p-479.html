A pebble falls from a 64 ft tall building. Its height (in ft) above the ground \(t\) seconds after it drops is given by the function \(y=f(t)=64-16t^2\text{.}\).

  1. What is \(n\) (the number of inputs)?

  2. What is \(m\) (the number of outputs)?

  3. Construct a graph of this function.

  4. How many dimensions do you need to graph this function?
