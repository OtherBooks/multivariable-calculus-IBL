The bounds of the integral \begin{equation*} \ds\int_{-3}^{3} \left(\int_0^{\sqrt{9-y^2}}(9-x^2-y^2) dx\right) dy \end{equation*} describe a region \(R\) in the plane, namely \begin{equation*} -3\leq y\leq 3 \text{ and } 0\leq x\leq \sqrt{9-y^2}. \end{equation*}

  1. Draw this region \(R\) in the \(xy\)-plane.

  2. Then give bounds to describe the region alternately by first stating constants which trap \(x\) (so \(a\leq x\leq b\)) and then functions which trap \(y\) (so \(c(x)\leq y \leq d(x)\)).

  3. Use these new bounds to write an iterated integral \begin{equation*} \ds\int_{x=a}^{x=b} \left(\int_{y=c(x)}^{y=d(x)}9-x^2-y^2 dy\right) dx \end{equation*} that gives the exact same volume of the solid \(D\) from the previous problem.
