The projection is \(\proj_{\vec d}\vec F = \frac{10}{5}(2,-1) = (4,-2)\text{.}\). This is the parallel component \(\vec F_{|| \text{ to } \vec d} =(4,-2)\text{.}\). To get the orthogonal component, we know that \(\vec F = \vec F_{|| \text{ to } \vec d} + \vec F_{\perp \text{ to } \vec d}\text{.}\). Vector subtraction gives \(\vec F_{\perp \text{ to } \vec d} = \vec F -\vec F_{|| \text{ to } \vec d} = (0,-10)-(4,-2) = (-4,-8)\text{.}\). We now write \begin{equation*} \vec F = (4,-2) + (-4,-8). \end{equation*} in-context
