Since a partial derivative is a function, we can take partial derivatives of that function as well. If we want to first compute a partial with respect to \(x\text{,}\), and then with respect to \(y\text{,}\), we would write one of \begin{equation*} f_{xy}=\ds\frac{\partial}{\partial y}\frac{\partial}{\partial x}f = \frac{\partial}{\partial y}\frac{\partial f}{\partial x} = \frac{\partial^2 f}{\partial y \partial x}. \end{equation*}
