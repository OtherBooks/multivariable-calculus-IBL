We can write this in vector notation as \(\begin{pmatrix}x\\y \end{pmatrix} =\begin{pmatrix}r\cos\theta\\ r\sin\theta \end{pmatrix}\text{.}\). This is a vector equation in which you input polar coordinates \((r,\theta)\) and get out Cartesian coordinates \((x,y)\text{.}\). So you input one thing to get out one thing, which means that we have a function. We could write \(\vec T(r,\theta) = (r\cos\theta,r\sin\theta)\text{,}\), where we've used the letter \(T\) as the name of the function because it is a transformation between coordinate systems. To emphasize that the domain and range are both two dimensional systems, we could also write \(T:\mathbb{R}^2\to\mathbb{R}^2\text{.}\). In the next chapter, we'll spend more time with this notation.
