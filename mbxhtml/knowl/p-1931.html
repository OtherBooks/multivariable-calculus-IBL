Let's start by showing how Green's theorem extends to 3D. A vector field in the plane \(\vec F = \left(M,N\right)\) can be extended to 3D by writing \(\vec F = \left(M,N,0\right)\text{.}\). The curl is \(\text{ curl } \vec F = \left\lt 0,0,N_x-M_y\right>\text{.}\). A normal vector to a surface \(S\) in the \(xy\) plane is \(\vec n = \left\lt 0,0,1\right>\text{.}\). We can then write Green's theorem as \begin{equation*} \oint_C\vec F\cdot d\vec r = \oint_C Mdx+Ndy = \iint_S \text{ curl } \cdot \vec n d\sigma = \iint_R \left(N_x-M_y\right) dA, \end{equation*} where \(C\) is a simple closed curve in the plane and \(S=R\) is the planar region that is inside \(C\text{.}\).
