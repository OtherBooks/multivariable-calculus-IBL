Exercise14.1.3

Consider the iterated integral \begin{equation*} \int_{-1}^1\int_0^{1-x^2}\int_0^{y} dzdydx. \end{equation*}

The bounds for this integral describe a region in space which satisfies the 3 inequalities \(-1\leq x\leq 1\text{,}\) \(0\leq y\leq 1-x^2\text{,}\) and \(0\leq z\leq y\text{.}\)

  1. Draw the solid domain \(D\) in space described by the bounds of the iterated integral.

There are 5 other iterated integrals equivalent to this one.

  1. Set up the integral that use the differential ordering: \(dydxdz\)

  2. Set up the integral that use the differential ordering: \(dxdzdy\text{.}\)

We'll create the other 3 in class (though you are welcome to include them as part of your presentation)
