EXTRA 1 You'll want to memorize this result. If you use the spherical parametrization \begin{equation*} \vec r(\phi,\theta) = (a\sin\phi\cos\theta,a\sin\phi\sin\theta,a\cos\phi), \end{equation*} show that \begin{equation*} d\sigma = (a^2|\sin\phi|)d\phi d\theta= (a^2\sin\phi) d\phi d\theta, \end{equation*} where we can ignore the absolute values if we require \(0\leq \phi\leq \pi\text{.}\). Along the way, you'll show that \begin{equation*} \vec r_\phi\times \vec r_\theta = a^2\sin \phi (\sin\phi\cos\theta,\sin\phi\sin\theta,\cos\phi). \end{equation*}
