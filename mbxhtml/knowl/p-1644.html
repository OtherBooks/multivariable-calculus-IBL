We can either set up the bounds with \(x\) on the inside, or \(y\) on the inside. We'll get the same answer. When we set up the integral with bounds, we call it an iterated integral, and write. \begin{equation*} \ds \int_a^b \int_{c(x)}^{d(x)}f(x,y)dydx \text{ or } \ds \int_c^d \int_{a(y)}^{b(y)}f(x,y)dxdxy. \end{equation*}
